# The Agentic Harness

A book about what it takes to put a language model to work in the real world — the loop, the tools, the context, the scaffolding around the model.

## Build

```
npm install
npm run all       # PDF + EPUB
npm run pdf       # PDF only
npm run kindle    # EPUB only
```

Chapters live in `src/` as Markdown. Output lands in `output/`.
