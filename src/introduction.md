# Introduction

### *The model is only half the story.*

This book is about the other half.

When people talk about AI, they talk about the model. Which one is best. Which one is cheapest. Which one can hold the most in its head at once. That is the conversation everyone is having, and it is a real conversation, and it matters. But it is not the conversation this book is about.

This book is about what happens around the model. The loop that calls it. The tools it can reach for. The context it is given and the context it is not. The way its output gets checked, trusted, acted on, or thrown away. All the scaffolding that turns a text predictor into something that can actually do a job.

I am going to call all of that the harness. It is not a standard term. People use different words for it — the agent loop, the runtime, the orchestration layer, the glue. The point is that it is a real thing, and it is doing more work than most people give it credit for.

---

## What this book is not

This is not a tutorial. I am not going to teach you how to build an agent from scratch in a weekend. There are plenty of places to learn that, and the details change faster than a book can keep up with.

This is not a survey of every framework. I have opinions about some of them, and they will show up, but the goal is not to rank tools.

This is not a prediction about where the field is going. I do not know where it is going. Anyone who tells you they do is selling something.

## What this book is

This is one person's attempt to describe what it actually feels like to build and use these systems. What breaks. What surprises you. What the hard parts really are, once you get past the demo.

The harness is the thing I think about most, and the thing I think gets talked about least. That is the gap I am trying to close.

---

## How to read it

Start anywhere. The chapters are meant to stand on their own. If one topic grabs you, go there first. If you want the full picture, the order is roughly: what a harness is, what it does, what makes it hard, and what I think matters.

There is no test at the end.
